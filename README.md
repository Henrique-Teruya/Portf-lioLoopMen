# Portfolio LoopMen 🎨🎬

Este é o repositório do portfólio digital do artista visual e profissional do audiovisual **João Vitor Alves**, conhecido como **LoopMen**. O projeto apresenta uma interface moderna com estética urbana e "street", refletindo a identidade criativa do artista.

## 📌 O que é?
Um web portfólio responsivo desenvolvido com tecnologias front-end puras. Ele serve como uma vitrine para trabalhos de:
*   **Ilustração:** Estilo cartoon, cenários e personagens.
*   **Audiovisual:** Edição de vídeo, correção de cor (Color Grading) e animação/motion.
*   **Design Gráfico:** Identidade visual e estampas.
*   **Fotografia:** Still e operação de câmera.

## 👤 Para quem foi feito?
O portfólio foi desenvolvido para o artista **João Vitor Alves (LoopMen)**. O objetivo é centralizar sua trajetória, exibir seus principais projetos (como parcerias com RapBox, Instituto KondZilla e produções cinematográficas) e facilitar o contato de possíveis clientes e parceiros.

## 🛠️ Quem fez?
Este projeto foi desenvolvido por **Henrique Teruya de Queiroz**. 
A construção focou em criar uma experiência de usuário fluida, com foco em performance e uma navegação visualmente impactante que utiliza elementos como:
*   **Menu Overlay:** Navegação limpa e intuitiva.
*   **Ticker/Carrossel:** Exibição dinâmica de artes e ilustrações.
*   **Lightbox:** Visualização detalhada das artes em alta resolução.

## 🎯 Por que foi feito?
O projeto nasceu da necessidade de criar uma presença digital profissional que fugisse do "comum". A ideia foi unir a funcionalidade de um currículo técnico com a expressividade de uma galeria de arte, utilizando uma paleta de cores vibrante (neon/azul) e um layout que conversa diretamente com a cultura Hip Hop e a estética das ruas, que são as bases da arte do LoopMen.

*   CSS3 (Flexbox e Grid para layout responsivo)
*   JavaScript Vanilla (Interações, Menu e Lightbox)
